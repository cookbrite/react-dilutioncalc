## React Tape

A boilerplate for React using Tape for testing.

### Getting Started

1. clone this repo
2. cd into folder
3. `npm install`
4. `npm start`
5. open http://localhost:3000

### Testing

```
npm test
```

### Linting

```
npm run lint
```

### Build for Production

```
npm run deploy
```
